# Type System

The [type system](#type-system) is a logical system for binding the rules of memory usage and semantic
possibilities to evry object in the program.

## Type Safety

## Type Checking

The Mjolnir type system is statically typed, which means that types are defined, at compile time and
not determined at runtime.

## Nominally Typed

The Mjolnir type system is nominally typed, which means that types are compared by definition and not by any conceptual characteristics of their definition.

Two structure types which have identical, yet distinct, definitions are not equivalent types.
