# If

## If Statement

Conditionally executes another statement.

## Syntax

Simple `if` statement.

```mj
if (...) {
    ...
}
```

Simple `if-else` statement.

```mj
if (...) {
    ...
} else {
    ...
}
```

Simple `if-else-if` statement.

```mj
if (...) {
    ...
} else if (...) {
    ...
}

if () {
    ...
} else do {
    ...
} while () {
    ...
} catch {
    ...
}
```
