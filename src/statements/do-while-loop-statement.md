# `do-while-loop` Statement

A `do-while` loop statement combines the properties of the `do` and `while` loop statements to
create a middle tested loop.

## Syntax

## Basic `do-while` loop

This statement 

```mj
do {
    // loop body
} while (...) // loop expression
```

## Middle-tested `do-while` loop

```mj
do {
    // always run
} while (...) {
    ...
}
```
